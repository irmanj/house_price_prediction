# 🏠 House Price Prediction

Aplikasi Machine Learning untuk memprediksi harga rumah berdasarkan luas dan jumlah kamar.

## 🚀 Features
- Input data rumah (luas rumah, jumlah kamar, usia rumah, lokasi)
- Prediksi harga otomatis
- Evaluasi model (MAE & RMSE)

## 📊 Model Performance
- MAE: 7.35
- RMSE: 7.35
- R2 Score: nan

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

## ▶️ Run
```bash
python main.py
streamlit run app.py
